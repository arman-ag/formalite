### Theming

because all input in Formalite is from MUI so we can use <code>ThemeProvider</code> to set theme and set color and size to components.

<Canvas>
  <Story 
    name="ThemeProvider"
    parameters={{ docs: { source: { 
        code: ThemeTextView,
        language: "javascript",
        type: "auto"
      }} 
    }}
    args={{ 
     layoutProps: {
         xs: 12,
       },
       inputProps: {
         label: "Title Input",
         helperText: "Helper text",
         placeholder: "some other title",
         onChange: (value) => {
           console.log(value);
         },
       },
       theme:{
          shape:{
            borderRadius: 100
          },
          palette:{
            primary:{
              main :"#f00"
            }
          }
       }
    }}
    >
      {TextViewTemplate.bind({})}
  </Story>
</Canvas>

### Prop Customization

all Formalite inputs takes MUI props with <code>inputProps</code>. 

<Canvas>
  <Story 
    name="inputProps"
    parameters={{ docs: { source: {

     layoutProps: {
         xs: 12,

           endAdornment: <InputAdornment position="end" style={{color:"#ed6c02"}}>kg</InputAdornment>,
         },                  
         minRows:3,
         size:"big",
         multiline:true,

           console.log(value);
         },
       },
       theme:{}
    }}
