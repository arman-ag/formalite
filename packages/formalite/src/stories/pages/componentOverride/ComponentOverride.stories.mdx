# Component Overriding

You can override all input component with `components` props in Formalite

## Usage

In this usage developer changes backgroundColor of TextView Component.

```js
import { Formalite, FormaliteTextView } from "@novin-dev/formalite";

export const TextView = () => {
 return (
   <Formalite
     lang="en"
     formString={someFromString}
     initialValues={someIniValues}
     validationSchema={someValidation}
     formRef={formRef}
     onSubmit={(values) => {
       console.log(values);
     }}
     components={{
       TextView : props => (
         <div style={{ backgroundColor: '#042859' }}>
           <FormaliteTextView {...props} />
         </div>
      )
     }}
   />
 );
};
```
